#  UEFA Champions League Historical Data Scraper & Analysis

This repository contains the Python web scraping and data engineering scripts used to collect historical UEFA Champions League (UCL) knockout stage data from the 1992 season to the present.

##  Tools & Technologies
* **Language:** Python
* **Libraries:** Pandas, BeautifulSoup, Requests
* **Data Source:** Transfermarkt.com

##  The Project
The scripts in this repository perform the following actions:
1.  **Web Scraping:** Automatically extract match details, stadium attendances, and referee data.
2.  **Data Engineering:** Apply custom logic to separate and identify exact penalty shootout scores, isolating them from regular/extra-time goals.
3.  **Hypothesis Testing:** Prepare distinct data populations to analyze the statistical impact of the abolished "Away Goal Rule".

##  Dataset
The fully cleaned and structured dataset generated by these scripts is published on Kaggle. 
You can view and download the dataset here: **[https://www.kaggle.com/datasets/emirhansevinc/ucl-all-knockout-games-1992-2026]**
